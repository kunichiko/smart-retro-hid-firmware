Mimic X Firmware Update

Mimic X (CH32X035) のファームウェアを Web ブラウザから書き込むページです。 プロジェクト: MimicX-firmware

準備

CH32X035 の BOOT ボタンを押しながら USB ケーブルを接続する（または BOOT を押しながら RESET を押して離す）
このページで「デバイスに接続」ボタンを押す
「QinHeng Electronics 製のデバイス」(または「WinChipHead 製の不明なデバイス」) を選択する
基板に合うバリアントとバージョンを選択して「書き込み」を押す
書き込み完了後、USB ケーブルを抜き差しして通常モードで起動する
Chromium 系ブラウザ（Chrome, Edge）が必要です。Safari / Firefox は WebUSB に対応していません。
